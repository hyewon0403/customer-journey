## 고객 구매 여정 분석

### 프로젝트 소개
인터넷 쇼핑몰에서 고객의 이탈률, 종료율이 증가하고 있어 이를 해결하기 위해 **구매 여정 분석을 통해 고객이 이탈하는 페이지를 탐색**했습니다.
이탈률과 종료율을 기준으로 개선이 필요한 단일 페이지 및 사용자 여정 개선 포인트를 선정하였습니다.
